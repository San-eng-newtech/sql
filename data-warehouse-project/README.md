# Data Warehouse Project

## Overview
End-to-end Data Warehouse built using SQL following Medallion Architecture.
Based on the "Data with Baraa" 30-hour course.

## Architecture
Bronze → Silver → Gold

## Tech Stack
- SQL Server / SSMS
- T-SQL

## Layers
- **Bronze:** Raw data ingestion
- **Silver:** Cleaned & transformed data
- **Gold:** Business-ready tables & views

## Data Sources
- ERP System
- CRM System
